# 2025-26-MSIM4312_L03-Digital-assets-IRT-Dataset-Basic-Statistic-and-Test-Result
This repository provides a structured research dataset developed to study Innovation Resistance Theory (IRT) in the context of young adults’ resistance to exploring digital assets
The dataset captures multiple functional and psychological barriers:

1. Usage Barrier
2. Value Barrier
3. Risk Barrier
4. Traditional Barrier
5. Image Barrier
